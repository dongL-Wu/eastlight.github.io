    C++ 基础知识点

    构建C++程序

    每个C++程序有且只有一个主函数（main函数），main函数主要给操作系统返回一个int类型的值，且必须是int类型。

    返回值0表示成功，非0的值表示失败类型

    函数定义包括四部分 函数返回值 函数名 形参列表以及函数体

    int main()
{
    return 0;
}

    编译 运行程序

    命名空间

    如果C++程序中加载的两个库中都有同名的函数，编译器不知道该使用哪一个函数 这个时候就需要命名空间，即定义一个范围，可以帮助我们避免不经意的名字定义冲突，以及使用库中相同名字导致的冲突。

    
namespace namespace_name{
    //代码声明
}

    为了调用有命名空间的函数或变量，可借助作用域描述符::

    C++标准程序库中的所有标识符都被定义于一个名为std的namespace中

    i++ ++i的区别(仍需了解)

    i++是后缀递增，++i是前缀递增。 x=i++; // 先让x编程i的值，再让i+1 x=++i; // 先让i+1，再让x = i i++每次都会返回一个无用的临时对象，即每一次都要进行一个创建和销毁对象的操作。

    变量 对象区别

    对象是值一块能存储数据并具有某种类型的内存空间 变量是一个具名的 可供程序操作的存储空间

    引用和指针

    引用就是给对象起了另一个名字 结合 操作符&

    • 引用必须初始化

    • 对引用或者引用指向的对象进行操作，两者都会改变

    • 引用不是一个对象 不能定义为引用的引用

    int i = 0;
int &val = i;  //val指向i， 是i的另一个名字 但是没有内存空间
            // 引用必须初始化

    指针是指向另外一种类型的符合类型，存放指向对象的地址 ,多用在函数间传递参数

    • definition：声明符写成 *d的形式 其中d是变量名 如果在一条语句中定义了几个指针变量，每个指针变量前面都必须有 *
    int *ip1, *ip2; //ip1, ip2都是指向int型对象的指针
double dp, *dp2; //dp2是指向double型对象的指针， dp指向double型对象
    • 指针存放对象的地址， 获取对象的地址：取地址符 &
    int ival = 42;
int *p = &ival;    //p是指向int型对象的指针 p存放变量ival的地址
    • 指针访问对象：解引用符 * 仅适合于那些指向某些确定对象的有效指针
    int ival = 43;
int *p = &ival;
std::cout<<*p;
    • 赋值 即指针存放新的地址，指向其他对象（赋值改变的永远是等号左边的对象）
    pi = &ival; //pi存放的是ival的地址  pi指向ival
*pi = 0; //pi存放的地址没变 指向的对象变为0

    • void * ：可存放任何对象的地址

    • 空指针：

      • nullptr // int *p = nullptr;

      • 0 // int *p = 0;

      • NULL // // int *p = NULL;

    • 理解代码

    记住理解语句的原则：从右往左理解；（constexpr声明的指针是例外，P67页底） int ival = 42; 值42，赋给了变量名ival，这个ival是int型 int *p = &ival; 变量ival，取出它的地址，赋给了变量名p，这个p是一个指针，指针指向的是int型

    int i = 1;
//int *p = &i;
int *p;
p = &i; //两种表达效果是一样的

//p的值是i的地址，*p的值是i的地址所存储的值也就是1.

    两者的区别：

    • 指针本身是对象 允许对指针赋值和拷贝。而且可以在生命周期内先后指向几个不同的对象

    • 指针无需在定义时赋初值，如果没有被初始化，将拥有一个不确定的值

    数组指针和指针数组

    int *p[4] ;  //p是一个数组 数组的元素是指针
int (*p)[4] ;  //p是指针  指向的是数组

    if

    根据条件决定控制流

    int i = 0;
if (i<10)
    i++;

else if (i>15)
    i = i+2;

else
    i = i+3;

    break
